# Illustration (Human-AI Certified Archive)

This directory is the single archive location for human-readable collaboration artifacts between human and AI.

## Scope

- Flowcharts (Mermaid)
- Key process explanations
- Structured Q&A records
- Execution and decision notes that need long-term traceability

## Conventions

- Chinese is the primary file.
- English copies use the `*.en.md` suffix.
- Maintain paired files whenever possible:
  - `xxx.md`
  - `xxx.en.md`
