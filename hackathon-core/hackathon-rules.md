# AVE Claw Hackathon 2026

Last updated: 2026-04-13

---

## Background

AVE Claw is an AI capability platform for on-chain ecosystems, providing core Skill modules including asset monitoring, on-chain alerts, and automated trading strategy execution.

To encourage developers and teams to build practical applications based on AVE Skills, AVE is hosting this hackathon as part of the 2026 Hong Kong Carnival official agenda. Open to global participants.

---

## Tracks

Participants must build a fully functional application or tool based on AVE Claw, calling at least ONE of the following skill modules:

### 1. Monitoring Skills
- On-chain asset tracking
- Price alerts
- Anomaly detection
- Risk alerts

### 2. Trading Skills
- Strategy signal generation
- Automated trading execution
- Portfolio management
- Backtesting analysis

### 3. Complete Application Scenario (Combined)
- Combines both Monitoring AND Trading Skills
- End-to-end application scenario

---

## Submission Requirements

| Item | Requirement |
|------|-------------|
| Source Code | GitHub repository link |
| Documentation | Project docs including AVE Skill usage details |
| Demo Video | Functional demo video, max 5 minutes |
| Originality | Must be original work. No plagiarism or simple wrapping of existing open-source projects |
| Runnability | Must be actually runnable; judges may request live demo |

## Eligibility

- Individual or team (max 5 members per team)
- One person can only join one team
- No nationality/region restrictions, global participation welcome

| Dimension | Weight | Description |
|-----------|--------|-------------|
| **Innovation** | **30%** | Unique ideas or solutions, breakthrough innovation beyond existing approaches |
| **Technical Implementation** | **30%** | Code quality, architecture rationality, AVE Skill integration depth & proficiency, stability |
| **Practicality & Business Value** | **40%** | Solves real-world problems, has deployment potential and market viability |
